FlyeChat

FlyeChat

FlyeChat is a web-based, real time chat client, similar to Discord.

I originally started developing it at the start of 2021 in January, using Python with a client and server application, connected using the socketserver library. It worked quite well, even integrating Discord Rich Presence, however there was one major flaw: whenever I converted it to an exe for other people to try, Windows Defender would flag it as a virus.

It was this that ultimately drove me to make a web version. I firstly made a Python web version using Flask, however my HTML/CSS skills were not very good at this time. It was at this point NewGuitarWhoDis joined the project. Ever since, he has been responsible for making the great designs for FlyeChat. The Flask web version worked fairly well, but I felt limited by Python and wanted to try something new.

At this point, I decided to learn Microsoft's Asp.Net. I remade the whole system from scratch using Asp.Net, and it turned out pretty well.

However in December 2021, I decided I again wanted to recreate FlyeChat with a whole new design and feature set - which is now the current version of FlyeChat. For the first time, FlyeChat now wasn't just a texting platform, it now had voice chats with screensharing.

This latest version is nearly ready and ready to be released, but for now:

You can access the old version here: https://FlyeChat.com

ETS2/ATS Trackers

ETS2/ATS Trackers

FlyeTracker is a system to log your deliveries in the games Euro Truck Simulator 2 & American Truck Simulator. There is a downloadable tracker software which connects to the games and logs and sends your deliveries to the server. You can then view all your deliveries online in the web application.

I have also made several custom Drivers Hubs with trackers for VTCs.

You can FlyeTracker here: https://tracker.flye001.com
